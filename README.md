# <div align=center>nebula-org-chart</div>

#### 介绍
基于jquery的org-chart插件，实现org-chart的绘制。

希望大家都能参与进来。共同成长

#### 使用方法

直接将index.html 下载下来在浏览器中展示即可

#### 目前实现的功能

1. 支持json数据源的绘制

2. 支持收缩/展开
